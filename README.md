# Palettes in Sitecore
Palettes in Sitecore module implements a Colour Palette Management in Sitecore.

# Sitecore Packages
Sitecore packages contain:
1. **Data Templates**
   - Palette
   - Palette Theme
   - Theme Content Page
1. **Renderings**
   - Palette Styles
1. **Palette Themes** - pre-defined colour schemes
